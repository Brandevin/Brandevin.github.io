Classification of movies with the Bechdel database

Exploration, statistical analysis and
classification using the Bechdel movies database.

The Bechdel test is a measure of the representation of women in fiction. It asks whether a movie features at least two women who talk to each other about something other than a man. About half of all films meet these criteria

Using the data available on bechdeltest, the tendencies of the test are analysed under a statistical focus. Some of the following questions are addressed:

  • Has the ratio of movies that pass the Bechdel Test increased over time?
  • Is it possible to see a different ratio of movies that pass the test for different movie genres?
  • Are movies that pass the Bechdel better rated by the viewers?
  • Does the gender of the director influence the chance of a movie passing the test?

On the second part of the analysis, classification machine learning models are created to predict wheter or not a movie passes the test. It uses features such as director gender, release year, movie genre and others.

The models are created using sklearn's built-in methods to preprocess and classify the data, as well as tensorflow's utilities to build neural networks.
